Tentang Aplikasi

Aplikasi ini adalah sistem pelaporan sekolah (UKK) yang memungkinkan siswa membuat laporan, dan admin mengelola serta memberikan tanggapan atas laporan tersebut.

Peran Pengguna

🔐 Admin — Kelola laporan, kategori, berikan feedback 📝 Siswa — Buat & lihat laporan sendiri

Stack Teknologi

  • Framework: Laravel 11 (PHP 8.x)
  • Frontend: Blade Templates + Tailwind CSS
  • Database: MySQL
  • Auth: Laravel Breeze (session-based)
  • Build: Vite + PostCSS

Method: feedback() — Tanggapan Admin

Method ini digunakan admin untuk memberikan tanggapan dan mengubah status laporan.

public function feedback(Request $request, $id)
{
    // 1. Cari laporan, auto 404 jika tidak ada
    $laporan = Laporan::findOrFail($id);

    // 2. Update status dan feedback dari form
    $laporan->status = $request->status;
    $laporan->admin_feedback = $request->admin_feedback;

    // 3. Jika ditolak tapi feedback kosong → isi default
    if ($request->status == 'ditolak' && !$request->admin_feedback) {
        $laporan->admin_feedback = 'Laporan ditolak';
    }

    // 4. Simpan ke database
    $laporan->save();

    return back()->with('success', 'Tanggapan berhasil dikirim');
}

Laporan Model Model

Model utama yang merepresentasikan data laporan dari siswa.

class Laporan extends Model
{
    protected $fillable = [
        'user_id', 'kategori_id', 'admin_id',
        'lokasi', 'deskripsi', 'foto',
        'status', 'admin_feedback',
        'diterima_pada', 'ditanggapi_pada', 'selesai_pada'
    ];

    // Relasi ke User (pembuat laporan)
    public function user() {
        return $this->belongsTo(User::class);
    }

    // Relasi ke Kategori
    public function kategori() {
        return $this->belongsTo(Kategori::class);
    }

    // Relasi ke Tanggapan
    public function tanggapans() {
        return $this->hasMany(Tanggapan::class);
    }
}

Middleware

Middleware custom yang digunakan untuk authorization berbasis role.

RoleMiddleware Middleware

Middleware ini mengecek apakah user yang login memiliki role yang sesuai sebelum mengakses route tertentu.

class RoleMiddleware
{
    public function handle(Request $request, Closure $next, string $role)
    {
        if (!auth()->check() || auth()->user()->role !== $role) {
            return redirect('/')->with('error', 'Akses ditolak.');
        }

        return $next($request);
    }
}

Cara Penggunaan di Route

// Hanya Admin
Route::middleware(['auth', 'role:admin'])->group(function() {
    Route::resource('kategori', KategoriController::class);
});

// Hanya Siswa
Route::middleware(['auth', 'role:siswa'])->group(function() {
    Route::resource('laporan', LaporanController::class);
});

class StoreLaporanRequest extends FormRequest
{
    public function authorize(): bool
    {
        return true; // Cek di middleware/policy
    }

    public function rules(): array
    {
        return [
            'kategori_id' => ['required', 'exists:kategoris,id'],
            'lokasi'      => ['required', 'string', 'max:255'],
            'deskripsi'   => ['required', 'string'],
            'foto'        => ['nullable', 'image', 'max:2048'],
        ];
    }
}
